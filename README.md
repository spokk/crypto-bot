# Crypto Bot

A Telegram bot that provides up-to-date cryptocurrency price charts using CoinGecko data and QuickChart rendering.

## Features

- Get 7-day price charts for any supported cryptocurrency.
- Clean, dark-themed chart images.
- Fast and reliable data from CoinGecko.
- Easy deployment to Vercel.

## Example Command

Send a symbol to the bot (e.g.):

```
/btc
```

The bot will reply with a 7-day price chart for Bitcoin.
