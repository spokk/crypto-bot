<b>Crypto Bot Command List</b>

The following commands are available:
/btc - Bitcoin (BTC)
/eth - Ethereum (ETH)
/bnb - BNB (BNB)
/sol - Solana (SOL)
/steth - Lido Staked Ether (STETH)
/xrp - XRP (XRP)
/doge - Dogecoin (DOGE)
/ton - Toncoin (TON)
/ada - Cardano (ADA)
/shib - Shiba Inu (SHIB)
/avax - Avalanche (AVAX)
/wbtc - Wrapped Bitcoin (WBTC)
/trx - TRON (TRX)
/link - Chainlink (LINK)
/dot - Polkadot (DOT)
/matic - Polygon (MATIC)
/bch - Bitcoin Cash (BCH)
/ltc - Litecoin (LTC)
/uni - Uniswap (UNI)
/leo - UNUS SED LEO (LEO)
/near - NEAR Protocol (NEAR)
/dai - Dai (DAI)
/apt - Aptos (APT)
/atom - Cosmos (ATOM)
/fil - Filecoin (FIL)
/okb - OKB (OKB)
/cro - Cronos (CRO)
/imx - Immutable (IMX)
/pepe - Pepe (PEPE)

Type any command above (with a slash, e.g. /btc) to get the latest information for that cryptocurrency.
